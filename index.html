수집하는 정보

Nomio는 사용자의 개인정보를 수집하지 않습니다. 앱 사용에 필요한 최소한의 정보만 처리합니다.

  • 지출·수입·계좌 등 금융 데이터 기기 로컬 저장
  • 앱 설정 (언어, 통화, 테마) 기기 로컬 저장
  • 환율 조회 시 외부 API 호출 exchangerate-api.com
2

데이터 저장 위치

입력하신 모든 금융 데이터는 기기 내부 저장소에만 저장됩니다. Nomio 서버 또는 제3자 서버로 전송되지 않습니다.

  • 앱 삭제 시 모든 데이터 함께 삭제
  • 백업 기능 사용 시 사용자가 직접 선택한 위치(iCloud / Google Drive / 로컬)에 저장
3

광고 (Google AdMob)

Nomio는 무료 서비스 유지를 위해 Google AdMob 배너 광고를 표시합니다.

  • Google은 광고 ID를 사용하여 관심 기반 광고를 표시할 수 있습니다
  • Android: 설정 → 개인정보 보호 → 광고 → 광고 ID 재설정 또는 맞춤 광고 거부
  • iOS: 설정 → 개인정보 보호 → Apple 광고
  • 자세한 내용: Google 개인정보처리방침
4

생체 인증

앱 잠금 기능 사용 시 기기의 생체 인증(Face ID, 지문)을 사용합니다.

  • 생체 정보는 기기 보안 칩에서만 처리됩니다
  • Nomio는 생체 데이터에 접근하거나 저장하지 않습니다
5

아동 보호

Nomio는 13세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 수집하지 않습니다.

Information We Collect

Nomio does not collect personal information. We process only the minimum data necessary for app functionality.

  • Financial data (expenses, income, accounts) stored locally
  • App settings (language, currency, theme) stored locally
  • Exchange rate API calls exchangerate-api.com
2

Data Storage

All your financial data is stored only on your device. It is never sent to Nomio servers or any third-party server.

  • All data is deleted when the app is uninstalled
  • Backups are saved to a location you choose (iCloud / Google Drive / local)
3

Advertising (Google AdMob)

Nomio displays Google AdMob banner ads to keep the app free.

  • Google may use your Advertising ID to show interest-based ads
  • Android: Settings → Privacy → Ads → Reset Advertising ID or opt out
  • iOS: Settings → Privacy → Apple Advertising
  • More info: Google Privacy Policy
4

Biometric Authentication

When app lock is enabled, your device's biometric authentication (Face ID, fingerprint) is used.

  • Biometric data is processed only by the device's secure hardware
  • Nomio never accesses or stores raw biometric data
5

Children's Privacy

Nomio is not directed at children under 13 and does not knowingly collect personal information from children.

Erhobene Daten

Nomio erhebt keine personenbezogenen Daten. Wir verarbeiten nur die minimal notwendigen Daten für die App-Funktionalität.

  • Finanzdaten (Ausgaben, Einnahmen, Konten) lokal gespeichert
  • App-Einstellungen (Sprache, Währung, Design) lokal gespeichert
  • Wechselkurs-API-Anfragen exchangerate-api.com
2

Datenspeicherung

Alle Ihre Finanzdaten werden ausschließlich auf Ihrem Gerät gespeichert. Sie werden nie an Nomio-Server oder Dritte übertragen.

  • Alle Daten werden beim Deinstallieren der App gelöscht
  • Backups werden an einem von Ihnen gewählten Ort gespeichert (iCloud / Google Drive / lokal)
3

Werbung (Google AdMob)

Nomio zeigt Google AdMob-Bannerwerbung an, um die App kostenlos zu halten.

  • Google kann Ihre Werbe-ID für interessenbezogene Werbung verwenden
  • Android: Einstellungen → Datenschutz → Werbung → Werbe-ID zurücksetzen
  • iOS: Einstellungen → Datenschutz → Apple-Werbung
  • Mehr Infos: Google Datenschutzrichtlinie
4

Biometrische Authentifizierung

Bei aktivierter App-Sperre wird die biometrische Authentifizierung Ihres Geräts verwendet (Face ID, Fingerabdruck).

  • Biometrische Daten werden nur vom sicheren Hardware-Chip des Geräts verarbeitet
  • Nomio greift nie auf biometrische Rohdaten zu und speichert diese nicht
5

Kinderschutz

Nomio richtet sich nicht an Kinder unter 13 Jahren und erhebt wissentlich keine personenbezogenen Daten von Kindern.
